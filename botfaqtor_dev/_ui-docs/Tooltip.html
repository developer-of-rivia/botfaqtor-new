Tooltip (подсказка)

Чтобы вызвать подсказку при наведении на нужный элемент, необходимо добавить на этот элемент специальные data-атрибуты: `data-tippy` и `data-tippy-content` со значением, это значение отобразится в качестве контента тултипа.

В данном примере мы сделали так, чтобы при наведении на заголовок карточки у нас появлялся тултип. Добавить тултип таким образом можно к большинству элементов.

Управление компонентом

Тултип может управляться из двух мест:

1) из javascript

2) с помощью data-атрибутов

При этом важно понимать, что настройки из js применяются для всех тултипов на проекте, а data-атрибуты являются удобным способом настройки конкретного тултипа (а также перебивают настройки из js).

Настройка `trigger` - является аналогом дата атрибутам `data-tippy-trigger`
Настройка `placement` - является аналогом дата атрибутам data-tippy-placement
Соответственно
javascript: *
html: data-tippy-*
