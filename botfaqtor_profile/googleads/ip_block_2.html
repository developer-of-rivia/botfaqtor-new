Блокировка IP

Не хотите тратить деньги на бесполезные клики? В Google Ads возможно заблокировать показ рекламы для списка IP-адресов. В настоящее время верхний предел, наложенный Google, составляет 500 IP.

На этой странице вы можете выбрать рекламные кампании, в которых мы автоматически выставим блокировку показов для 500 наиболее опасных для вас ботов.

Примечание. Исключение IP-адресов недоступно для видеокампаний, кампаний для приложений, умных кампаний в контекстно-медийной сети и кампаний Discovery.
